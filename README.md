# Lang Rush Client

## Getting started

```bash
   # Install packages
   $ pnpm install

   # configure .env
   $ cp .env.example .env
```

## Server

Set up [server](https://github.com/lang-rush/server) for this application

## Running the app

```bash
   # Run in development mode
   $ pnpm dev

   # Preview the production
   $ pnpm build
   $ pnpm preview
```
